# Пользователи решения

Приложение нацелено для взаимодействия (покупка/продажа) между покупателями и продавцами.

Продуктовые команды отвечают за отдельно взятый изолированный функционал (разработчики, системные/бизнес-аналитики,
менеджеры, тестировщики)

В команду сопровождения продукта входят люди, отвечающие за разбор и оперативное решение иницидентов.

Команда аналитики (возможно отдельно продуктая команда) отвечает за анализ данных за определенный промежуток времени
для выстраивания маркетинговый целей и выяснения пожеланий пользователей.
